5. หลักฐานและการไขคดี
เบาะแสสำคัญ :
  • น้ำตาของฮิโตมิไหลออก "ด้านข้าง" (เพราะนั่งกลับหัวไม่ได้นั่งปกติ)
  • ผู้หญิงที่เล่นยิมนาสติกจะมีสมดุลร่างกายและกล้ามเนื้อขาดี
  • ความเร็วลมปะทะขณะรถไฟเหาะวิ่ง
กลอุบาย (Trick) :
กดเพื่อดูทริคฆาตกรรม

ใช้อุปกรณ์ในกระเป๋า (ตะขอ + ลวดเปียโน/สร้อยมุก) เกี่ยวรางรถไฟและคล้องคอเหยื่อ โดยใช้ขาเกี่ยวที่กั้นนิรภัยเพื่อทรงตัวโหนไปด้านหลัง อาศัยแรงความเร็วรถไฟกระชากศีรษะเหยื่อ

6. บทสรุปคดี
วิธีเปิดโปง :ชี้ให้ดูรอยช้ำที่ขา และให้ลองทำท่าทางจำลอง
ผลลัพธ์ :จับกุม (สารภาพ)

"ฉากท้าย : ชินอิจิแยกทางกับรันเพื่อตามชายชุดดำไป"
7. ความเชื่อมโยงเนื้อเรื่องหลัก
องค์กรชุดดำ :

ปรากฏตัวครั้งแรก (ยิน & วอดก้า) ทำธุรกรรมเถื่อน (ค้าอาวุธ/แบล็คเมล์)

พัฒนาการ :

ชินอิจิถูกตีหัวและกรอกยาพิษ APTX 4869 จนตัวหดเล็กลง

Manga vs Anime : ในมังงะชินอิจิโดนวอดก้าตีด้วยไม้ แต่ในอนิเมะเป็นกระบองเหล็ก

Location : Tropical Land ได้แรงบันดาลใจจาก Disneyland + Shima Spain Village

คดีลักพาตัวลูกสาวประธานบริษัท

Company President's Daughter Kidnapping Case

社長令嬢誘拐事件 (Shachō Reijō Yūkai Jiken)

1. ข้อมูลทั่วไป
วันที่ออกอากาศ :15 ม.ค. 1996
ซีซัน / ปี :Season 1 (1996)
ความยาว :24 นาที
Source Type :Manga-adapted
อ้างอิงมังงะ :Vol.1 File 2-5
2. โครงสร้างคดี
ประเภทคดีหลัก :ลักพาตัว (Kidnapping)
จำนวนคดี :1 (ซ้อนแผน)
ลักษณะคดี :วางแผนล่วงหน้า (ปลอม) -> ฉับพลัน (จริง)
แรงจูงใจ :เงิน (คนร้ายจริง) / เรียกร้องความสนใจ (ลูกสาว)
ระดับความซับซ้อน : (ต่ำ)
3. สถานที่และเวลา
สถานที่หลัก :คฤหาสน์ทานิ & โรงแรม
สถานที่ย่อย :โกดังร้าง / โรงเรียน
ช่วงเวลา :กลางวัน -> กลางคืน
สภาพแวดล้อม :เมือง / ย่านโรงงาน
4. ตัวละครที่เกี่ยวข้อง
เหยื่อ (Victim)
อากิโกะ ทานิ
ลูกสาวประธาน

ชายร่างยักษ์
โจรเรียกค่าไถ่ (และพ่อบ้านอาโซ)
ผู้ว่าจ้าง
ประธานทานิ
พ่อของเหยื่อ
ตัวละครหลัก & พิเศษ
เอโดงาวะ โคนัน (New) โมริ รัน โมริ โคโกโร่ ดร.อากาสะ จัมโบ้ (สุนัข)
5. หลักฐานและการไขคดี
เบาะแสสำคัญ :
  • จัมโบ้ (สุนัข) : มีนิสัยเชื่องกับอากิโกะมาก (คนร้ายคิดว่าดุ)
  • เวลาที่หายไป : ไม่มีพยานเห็นคนแปลกหน้าเข้าออกในช่วงเวลาเกิดเหตุ (เพราะซ่อนในบ้าน)
  • วิวจากหน้าต่าง : อากิโกะโทรมาบอกว่าเห็น "โรงเรียน" และ "ปล่องไฟ" (ใช้ระบุพิกัดโกดัง)
กลอุบาย (Trick) :
กดเพื่อดูทริค

แผนซ้อนแผน : เดิมทีลูกสาวร่วมมือกับพ่อบ้านแกล้งถูกลักพาตัว (Fake) โดยซ่อนตัวในโรงแรมเพื่อเรียกร้องความสนใจ แต่ถูกคนร้ายตัวจริงบุกเข้ามาลักพาตัวไปจริงๆ (Real)

7. ความเชื่อมโยงเนื้อเรื่องหลัก
องค์กรชุดดำ :

ไม่มีบทบาทในตอนนี้

พัฒนาการ :

ชินอิจิกลายเป็น "เอโดงาวะ โคนัน" (เอาชื่อมาจาก สันหนังสือ) และย้ายเข้าไปอยู่สำนักงานนักสืบโมริ

8. เกร็ดข้อมูล

Gadgets : ตอนนี้โคนันยังไม่มีอุปกรณ์ไฮเทค (หูฟัง/นาฬิกา) ต้องใช้ไหวพริบล้วนๆ

Name Origin : ชื่อ "เอโดงาวะ โคนัน" มาจาก "เอโดงาวะ รัมโป" และ "อาเธอร์ โคนัน ดอยล์"

EPISODE JP : 3 / TH : 3

คดีฆาตกรรมในห้องปิดตายของยอดดารา

An Idol's Locked Room Murder Case

アイドル密室殺人事件 (Aidoru Misshitsu Satsujin Jiken)

1. ข้อมูลทั่วไป
วันที่ออกอากาศ :22 ม.ค. 1996
ซีซัน / ปี :Season 1 (1996)
ความยาว :24 นาที
Source Type :Manga-adapted
อ้างอิงมังงะ :Vol.1 File 6-9
2. โครงสร้างคดี
ประเภทคดีหลัก :ฆาตกรรม (Murder)
จำนวนคดี :1
ลักษณะคดี :ห้องปิดตาย (Locked Room)
แรงจูงใจ :ความเข้าใจผิด (ความรัก)
ระดับความซับซ้อน : (กลาง)
3. สถานที่และเวลา
สถานที่หลัก :แมนชั่นของโอกิโนะ โยโกะ
จุดเกิดเหตุ :ห้องนั่งเล่น
ช่วงเวลา :กลางวัน
สภาพแวดล้อม :ในห้องพักอาศัย
4. ตัวละครที่เกี่ยวข้อง
เหยื่อ (Victim)
ฟูจิเอะ อากิโยชิ
แฟนเก่าโยโกะ
คนร้าย (Culprit) กดเพื่อดูเฉลย
ฟูจิเอะ อากิโยชิ
ฆ่าตัวตายเพื่อใส่ร้าย (Suicide)
ผู้ต้องสงสัย
โอกิโนะ โยโกะ ยามากิชิ เออิจิ (ผู้จัดการ) อิเคซาวะ ยูโกะ (คู่แข่ง)
ตัวละครหลัก & พิเศษ
โคนัน รัน โคโกโร่ สารวัตรเมงูระ โอกิโนะ โยโกะ (New)
5. หลักฐานและการไขคดี
เบาะแสสำคัญ :
  • ก้อนน้ำแข็ง : รอยเปียกบนพื้นห้อง (เกิดจากน้ำแข็งละลาย)
  • อุณหภูมิห้อง : แอร์ถูกเปิดแรงสุดเพื่อเร่งให้น้ำแข็งละลาย
  • เส้นผม : ผมที่พบไม่ใช่ของโยโกะ แต่เป็นของยูโกะ (ที่มีลักษณะคล้ายกัน)
กลอุบาย (Trick) :
กดเพื่อดูทริค

น้ำแข็งสังหาร : ผู้ตายนำก้อนน้ำแข็งมาวาง เอาด้ามมีดฝังในหลุมที่ขุดบนพื้น แล้วตั้งมีดในน้ำแข็ง จากนั้นกระโดดทิ้งตัวลงมาเสียบมีดเอง เพื่อสร้างสถานการณ์ว่าเป็นคดีฆาตกรรม

วิธีไขคดี :
ใช้หูกระต่ายเปลี่ยนเสียงเลียนเสียงโมริ โคโกโร่ (หลังจากเตะที่เขี่ยบุหรี่ใส่จนสลบ)
หน้าฉาก :
โมริ โคโกโร่ (นิทราครั้งแรก? - ยังไม่เชิงนิทรา แค่สลบแล้วโคนันแอบพูด)
6. บทสรุปคดี
วิธีเปิดโปง :โคนัน (เสียงโคโกโร่) อธิบายทริคน้ำแข็ง
ผลลัพธ์ :ปรับความเข้าใจ
บทสรุป :เป็นการฆ่าตัวตายเพราะเข้าใจผิดว่าถูกทิ้ง
"ไดอารี่ที่พบเฉลยความจริงอันน่าเศร้าของความรัก"
7. ความเชื่อมโยงเนื้อเรื่องหลัก
องค์กรชุดดำ :

ไม่มีบทบาทในตอนนี้

พัฒนาการ :

ดร.อากาสะ มอบ "หูกระต่ายเปลี่ยนเสียง" ให้โคนันเป็นครั้งแรก และโคนันใช้มันไขคดีในนามโคโกโร่เป็นครั้งแรก (ต้นกำเนิดโคโกโร่นิทรา)

8. เกร็ดข้อมูล

Debut : เป็นตอนแรกที่ โอกิโนะ โยโกะ ปรากฏตัว และเธอกลายเป็นตัวละครสมทบที่มีบทบาทเรื่อยๆ (โดยเฉพาะเป็นขวัญใจของโคโกโร่)
