3. สถานที่และเวลา
  • สถานที่หลัก : เมืองเบกะ (ตามจุดต่างๆ ในแผนที่)
  • สถานที่สำคัญ : หอคอยโตโตะ (Tohto Tower) / โกดังริมน้ำ
  • ช่วงเวลาเกิดเหตุ : เย็น จนถึง กลางคืน
  • สภาพแวดล้อม : ในเมืองใหญ่ / แสงไฟนีออนยามค่ำคืน
4. ตัวละครที่เกี่ยวข้อง

ขบวนการนักสืบเยาวชน (รวมตัวครั้งแรก)

เอโดงาวะ โคนัน โยชิดะ อายูมิ ซึบุรายะ มิตซึฮิโกะ โคจิมะ เก็นตะ
กลุ่มอาชญากร
หัวหน้าแก๊ง : ดิโน่ (Dino)
องค์กร : แก๊งมาเฟียอิตาลี "Cabala"
ผู้กระทำผิด (Culprit)
คนร้าย : กดดูเฉลย
ดิโน่ และพวก

ขโมยเหรียญทองคำ 15,000 เหรียญ

5. หลักฐานและการไขคดี
เบาะแสสำคัญ (Key Clues)
  • แผนที่ปริศนา : กระดาษที่มีสัญลักษณ์แปลกๆ (หอคอย , หมวก , กางเกงใน , ร่ม)
  • คำว่า "ORO" : ข้อความที่เขียนไว้ในแผนที่ (ภาษาอิตาลีแปลว่า "ทอง")
  • แสงไฟนีออน : สัญลักษณ์ในแผนที่ตรงกับรูปร่างของ "ป้ายไฟนีออน" ตามตึกต่างๆ ในเวลากลางคืน
การถอดรหัส (Trick)
กดเพื่อดูวิธีถอดรหัส

สัญลักษณ์คือป้ายไฟ :

  • หอคอย = หอคอยโตโตะ
  • หมวก = ป้ายร้านขายหมวก
  • กางเกงใน = ป้ายแผนกชุดชั้นใน

จุดจบ : เมื่อมองจากหอคอยโตโตะ ตามแนวสัญลักษณ์ จะเห็นแสงไฟจากตึกสะท้อนลงแม่น้ำเป็นรูปปลา (หรือเงาตึก) ชี้พิกัดสมบัติ

วิธีไขคดีของโคนัน : สังเกตเห็นป้ายไฟนีออนที่มีรูปร่างเหมือนสัญลักษณ์ในแผนที่ และใช้ความรู้ภาษาอิตาลีแปลคำว่า ORO
ใครถูกใช้เป็น "หน้าฉาก" : ไม่มี (โคนันนำทีมเด็กๆ ไขปริศนาเอง)
6. บทสรุปคดี
วิธีจัดการคนร้าย :

โคนันเตะถุงใส่เหรียญทองคำร่วงลงมาทับคนร้ายจนสลบ (ใช้พลังขาโดยไม่มีรองเท้าเพิ่มพลัง)

  • ผลลัพธ์ : ถูกจับกุม
  • บทลงโทษ : ดิโน่และพรรคพวกถูกตำรวจจับกุม
"ฉากจบ : เด็กๆ ดีใจที่ไขคดีได้ แต่เสียดายที่สมบัติ (เหรียญทอง) เป็นของกลางต้องคืนตำรวจ"
7. ความเชื่อมโยงกับเนื้อเรื่องหลัก
เกี่ยวข้ององค์กรชุดดำ (ไม่มี)

ตอนนี้ยังไม่มีบทบาทขององค์กรชุดดำ

พัฒนาการตัวละคร
  • ขบวนการนักสืบเยาวชน : เป็นตอนแรกที่ เก็นตะ , มิตซึฮิโกะ และ อายูมิ รวมตัวกันไขคดีร่วมกับโคนันอย่างเป็นทางการ
8. เกร็ด & ข้อมูลเสริม
Language : คำว่า "ORO" ในภาษาอิตาลี แปลว่า "ทอง" (Gold) ซึ่งเป็นกุญแจสำคัญที่ทำให้โคนันรู้ว่าสมบัติคืออะไร
Culture/Location : Tohto Tower ในเรื่องมีต้นแบบมาจาก Tokyo Tower สถานที่ท่องเที่ยวชื่อดังของญี่ปุ่น

ผมชื่อเอโดงาวะ โคนัน ตัวจริงก็คือคุโด้ ชินอิจิ ที่โดนยาพิษจนกลายเป็นเด็ก ถึงจะตัวเล็กแต่สมองยังคงเป็นของยอดนักสืบคนเดิม
