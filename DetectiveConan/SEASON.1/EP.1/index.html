5. หลักฐานและการไขคดี
Case 1 : ข้ออ้างขาหัก
  • กลอุบาย : คนร้ายอ้างว่าขาหักต้องนั่งรถเข็น จึงไม่สามารถปีนหน้าต่างไปก่อเหตุได้
  • วิธีไขคดี (Logic) : ชินอิจิสังเกตพฤติกรรม และลองใจด้วยการเตะลูกโลกใส่
  • ข้อผิดพลาด : คนร้ายตกใจลุกขึ้นยืนหลบลูกโลก ทำให้ความแตกทันที
Case 2 : กลไกสังหารบนรถไฟเหาะ
เบาะแสสำคัญ (Key Clues)
  • น้ำตา : ไหลออกทาง "หางตาด้านข้าง" แสดงว่าตอนร้องไห้ร่างกายไม่ได้อยู่ในแนวตั้ง (กำลังห้อยหัวหรือเอียงตัว)
  • ร่างกาย : คนร้ายเป็นนักยิมนาสติก มีกล้ามเนื้อขาและสมดุลร่างกายดีเยี่ยม
  • แรงลม : ใช้ความเร็วลมปะทะขณะรถไฟวิ่งสร้างแรงดึงมหาศาล
กลอุบาย (Trick)
กดเพื่อดูทริคการฆาตกรรม

อุปกรณ์ : ตะขอเหล็ก, ลวดเปียโน/สายกระเป๋า, กระเป๋าถือ

ขั้นตอน : อาศัยจังหวะความมืดในอุโมงค์ ปลดล็อคที่นั่งตัวเอง แล้วใช้ขาเกี่ยวที่กั้นนิรภัยเพื่อทรงตัวโหนไปด้านหลัง จากนั้นนำตะขอที่มีลวดเกี่ยวไปที่รางรถไฟ และคล้องห่วงไปที่คอเหยื่อ อาศัยแรงความเร็วรถไฟกระชากศีรษะเหยื่อจนขาด

ข้อผิดพลาด : รอยช้ำที่ต้นขาของคนร้าย (เกิดจากการใช้ขาเกี่ยวที่กั้นนิรภัย)
ใครถูกใช้เป็น "หน้าฉาก" : ไม่มี (ชินอิจิไขคดีด้วยตัวเอง)
6. บทสรุปคดี
Case 1
  • วิธีเปิดโปง : ทำให้คนร้ายเผลอลุกขึ้นยืน
  • ผลลัพธ์ : ถูกจับกุม
Case 2
  • วิธีเปิดโปง : ชินอิจิชี้ให้ตำรวจดูรอยช้ำที่ขา และท้าให้จำลองท่าทาง
  • ผลลัพธ์ : คนร้ายสารภาพทั้งน้ำตาและถูกจับกุม
  • บทลงโทษ : ดำเนินคดีฆาตกรรม
"ฉากเฉลยสำคัญ : หลังจบไขคดี ชินอิจิสังเกตเห็นชายชุดดำทำตัวน่าสงสัย จึงบอกลาโมริ รัน และแอบสะกดรอยตามไป ซึ่งเป็นจุดเปลี่ยนของชีวิต"
7. ความเชื่อมโยงกับเนื้อเรื่องหลัก
เกี่ยวข้ององค์กรชุดดำ (มี)
  • การปรากฏตัวครั้งแรกของ ยิน (Gin) และ วอดก้า (Vodka)
  • มีการทำธุรกรรมแลกเปลี่ยนฟิล์มหลักฐานค้าอาวุธเถื่อนที่สวนสนุก
  • ชินอิจิถูกยินตีหัวจนสลบ
พัฒนาการตัวละคร & เบาะแส
  • ชินอิจิ : โชว์ความสามารถก่อนจะหายสาบสูญ
  • ยาพิษ APTX 4869 : ถูกนำมาใช้ครั้งแรกกับมนุษย์ โดยหวังผลให้ตายไร้ศพ แต่กลับทำให้ตัวหดเล็กลงแทน
ตอนเชื่อม / ปูเรื่อง

เป็นจุดเริ่มต้นของเรื่องราวทั้งหมด ปูพื้นฐานความสัมพันธ์ของชินอิจิกับรัน และสาเหตุที่ทำให้ชินอิจิต้องกลายเป็นเด็ก

8. เกร็ด & ข้อมูลเสริม
ข้อแตกต่างจากมังงะ :
  • ในมังงะ ชินอิจิโดนวอดก้าตีด้วย "ไม้" แต่ในอนิเมะเปลี่ยนเป็น "กระบองเหล็ก"
  • ฉากศีรษะขาดในมังงะจะวาดให้เห็นเลือดสาดกระเซ็นชัดเจนกว่า (อนิเมะมีแสงเซ็นเซอร์)
  • ในมังงะ ชินอิจิใส่ชุดนักเรียนไปเดท แต่ในอนิเมะใส่ชุดไปรเวท (เสื้อฮู้ดเขียว)
สถานที่จริง : สวนสนุก Tropical Land ได้รับแรงบันดาลใจผสมผสานจาก Disneyland และ Shima Spain Village ในญี่ปุ่น
จุดที่แฟนพูดถึง : ทรงผมของรันในตอนแรกยังไม่มี "เขา" (แหลมๆ) ที่ชัดเจนเหมือนตอนหลังๆ

ผมชื่อเอโดงาวะ โคนัน ตัวจริงก็คือคุโด้ ชินอิจิ ที่โดนยาพิษจนกลายเป็นเด็ก ถึงจะตัวเล็กแต่สมองยังคงเป็นของยอดนักสืบคนเดิม
