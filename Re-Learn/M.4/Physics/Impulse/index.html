โมเมนตัม และการชน

6.3 การดล

1. นิยามของการดล (\(I\))

การดล คือ ปริมาณที่บอกถึงผลของแรงที่กระทำต่อวัตถุในช่วงเวลาสั้นๆ

เขียนเป็นสมการได้ว่า :

\(\vec{I} = \sum \vec{F} \Delta t\)
หรือ
\(\vec{I} = \vec{F}_{avg} \Delta t\)
  • โดย \(\vec{F}_{avg}\) คือ แรงดลเฉลี่ย และ \(\Delta t\) คือ ช่วงเวลาที่แรงกระทำ
  • การดลเป็น ปริมาณเวกเตอร์ มีทิศทางเดียวกับแรงลัพธ์ที่มากระทำ
  • มีหน่วยเป็น นิวตัน วินาที (N·s) หรือ กิโลกรัม เมตรต่อวินาที (kg·m/s)

2. ทฤษฎีบทการดล-โมเมนตัม

(Impulse-Momentum Theorem)

การดลมีค่าเท่ากับ การเปลี่ยนแปลงโมเมนตัม ของวัตถุ

\(\vec{I} = \Delta \vec{P} = \vec{P}_f - \vec{P}_i\)
\(\vec{I} = m\vec{v} - m\vec{u} = m(\vec{v} - \vec{u})\)

( เมื่อ v คือความเร็วปลาย และ u คือความเร็วต้น )

3. การหาค่าการดลจากกราฟ

ถ้ามีกราฟความสัมพันธ์ระหว่าง แรง (\(F\)) กับ เวลา (\(t\)) :

  • พื้นที่ใต้กราฟ จะมีค่าเท่ากับ ขนาดของการดล (\(I\))
  • หลักการนี้ใช้ได้ทั้งกรณีแรงคงที่และแรงไม่คงที่
F
t
พื้นที่ = \(I\)

4. แรงดล (Impulsive Force)

คือ แรงลัพธ์ที่กระทำต่อวัตถุในช่วงเวลาสั้นๆ ซึ่งส่งผลให้โมเมนตัมของวัตถุเปลี่ยนไป

หาได้จากอัตราการเปลี่ยนโมเมนตัม :

\(\vec{F} = \frac{\Delta \vec{P}}{\Delta t} = \frac{m(\vec{v} - \vec{u})}{\Delta t}\)

(จากกฎข้อที่ 2 ของนิวตัน)

สรุปสั้นๆ

การดล
= แรง \(\times\) เวลา
= พื้นที่ใต้กราฟ \(F-t\)
= โมเมนตัมที่เปลี่ยนไป \((\Delta P)\)

ตัวอย่างการคำนวณ

ตัวอย่าง 6.5

ปล่อยไข่ไก่ดิบให้ตกบนฟองน้ำ ถ้าช่วงเวลาขณะไข่กระทบฟองน้ำจนหยุดเคลื่อนที่มีค่า 2.5 s และแรงเฉลี่ยที่ฟองน้ำกระทำต่อไข่มีค่า 0.1 N การดลที่เกิดขึ้นมีค่าเท่าใด

วิธีทำ

กำหนดให้ :
t = 2.5 s
F = 0.1 N
ขั้นตอนที่ 1 : ใช้สูตรการดล
\(I\) = F \cdot t = 0.1 \times 2.5 = 0.25 \(\text{N}\cdot\text{s}\)

คำตอบ

การดลที่เกิดขึ้นมีค่า 0.25 N \(\cdot\) s

ตัวอย่าง 6.6

ลูกบอลยางเคลื่อนที่ในแนวระดับไปทางขวาถูกตีสวนด้วยไม้ ทำให้ลูกบอลยางเคลื่อนที่กลับไปในทิศตรงข้าม โดยกราฟระหว่างแรงที่ไม้กระทำต่อลูกบอลยางกับเวลาขณะกระทบมีลักษณะดังรูป จงหาการดล

วิธีทำ

แนวคิด :

การดล (\(I\)) = พื้นที่ใต้กราฟ \(F-t\)

ขั้นตอนที่ 1 : คำนวณพื้นที่สามเหลี่ยม
\(I\) = \(\frac{1}{2} \times \text{ฐาน} \times \text{สูง}\) = \(\frac{1}{2} \times (3 \times 10^{-2}) \times 1000\) = \(\frac{1}{2} \times 0.03 \times 1000\) = 1.5 \times 10 = 15 \(\text{N}\cdot\text{s}\)

คำตอบ

การดลมีค่า 15 N \(\cdot\) s

ตัวอย่าง 6.7

ใช้ค้อนมวล 0.5 kg ตอกตะปูในแนวระดับ ในขณะที่ค้อนเริ่มกระทบหัวตะปู ค้อนมีขนาดความเร็ว 8.0 m/s หลังจากกระทบหัวตะปูแล้ว ค้อนสะท้อนกลับด้วยขนาดความเร็วเท่าเดิม ถ้าช่วงเวลาที่ค้อนกระทบหัวตะปูเป็น 1 ms แรงดลเฉลี่ยและการดลที่ค้อนกระทำต่อหัวตะปูเป็นเท่าใด

วิธีทำ

กำหนดให้ (ให้ทิศเข้าหาตะปูเป็น + , สะท้อนออกเป็น -) :
m = 0.5 kg
u = +8.0 m/s
v = -8.0 m/s
t = 1 ms = \(1 \times 10^{-3}\) s
ส่วนที่ 1 : หาการดล (\(I = \Delta P\))
\(I\) = m(v - u) = 0.5 [ (-8.0) - (+8.0) ] = 0.5 [ -16.0 ] = -8 \(\text{N}\cdot\text{s}\)
( ขนาดเท่ากับ 8 N·s )
ส่วนที่ 2 : หาแรงดลเฉลี่ย (\(F\))
\(F\) = \(\frac{I}{t}\) = \(\frac{-8}{1 \times 10^{-3}}\) = -8000 \(\text{N}\)
( ขนาดเท่ากับ 8000 N )

คำตอบ

การดลขนาด 8 N \(\cdot\) s

แรงดลเฉลี่ยขนาด 8,000 N

ตัวอย่าง 6.8
กระสุนปืนมวล 0.005 kg เคลื่อนที่เข้ากระทบแท่งไม้ที่ยึดแน่นกับพื้น ขณะเริ่มกระทบเนื้อไม้ กระสุนปืนมีอัตราเร็ว 400 m/s และสามารถทะลุเข้าไปในเนื้อไม้เป็นระยะ 0.1 m ถ้าแรงต้านของเนื้อไม้มีค่าคงตัว จงหา

ก. การดลที่เนื้อไม้กระทำต่อกระสุนปืน

ข. เวลาที่กระสุนปืนเคลื่อนที่ในเนื้อไม้

ค. แรงต้านของเนื้อไม้ที่กระทำต่อกระสุนปืน

วิธีทำ

กำหนดให้ :
m = 0.005 kg
u = 400 m/s
v = 0 m/s (หยุดในเนื้อไม้)
s = 0.1 m
ก. การดล (\(I\))
\(I\) = \(\Delta P = m(v - u)\) = 0.005(0 - 400) = -2 \(\text{N}\cdot\text{s}\)
( ขนาดเท่ากับ 2 N·s )
ข. เวลาที่เคลื่อนที่ (\(t\))

หาความเร็วเฉลี่ย \(v_{avg} = \frac{u+v}{2} = \frac{400+0}{2} = 200 \text{ m/s}\)

\(t\) = \(\frac{s}{v_{avg}}\) = \(\frac{0.1}{200}\) = 0.0005 \(\text{s}\) \((5 \times 10^{-4} \text{ s})\)
ค. แรงต้าน (\(F\))
\(F\) = \(\frac{I}{t}\) = \(\frac{-2}{0.0005}\) = -4000 \(\text{N}\)
( ขนาดแรงต้านเท่ากับ 4000 N )

คำตอบ

ก. การดลขนาด 2 N \(\cdot\) s

ข. เวลาที่ใช้ \(5 \times 10^{-4}\) s

ค. แรงต้านขนาด 4,000 N

คำถามตรวจสอบความเข้าใจ 6.3

ข้อ 1 การดลและแรงมีความสัมพันธ์กันหรือไม่ อย่างไร

คำตอบ : มีความสัมพันธ์กันตามสมการ \(\vec{I} = (\sum \vec{F}) \Delta t\)

ข้อ 2 แรงดล \(\vec{F}\) มีทิศทางเดียวกับปริมาณใดต่อไปนี้

ก. โมเมนตัม \(\vec{P}\)
ข. การเปลี่ยนโมเมนตัม \(\Delta \vec{p}\)
ค. การดล \(\vec{I}\)
ง. ความเร็ว \(\vec{v}\)
จ. การเปลี่ยนความเร็ว \(\Delta \vec{v}\)
ฉ. ความเร่ง \(\vec{a}\)

คำตอบ : แรงดล \(\vec{F}\) มีทิศทางเดียวกับ การเปลี่ยนโมเมนตัม \(\Delta \vec{p}\), การดล \(\vec{I}\), การเปลี่ยนความเร็ว \(\Delta \vec{v}\) และ ความเร่ง \(\vec{a}\) เพราะ \(\vec{F} = \frac{\Delta \vec{p}}{\Delta t} = \frac{\vec{I}}{\Delta t} = \frac{m\Delta \vec{v}}{\Delta t} = m\vec{a}\)

ข้อ 3 การเบรกรถจักรยานให้รถช้าลงจนหยุด กับการเบรกรถจักรยานให้รถหยุดทันที การดลในกรณีแรก มากกว่า เท่ากับ หรือน้อยกว่ากรณีหลัง จงอธิบาย

คำตอบ : เท่ากัน เพราะการดลเท่ากับโมเมนตัมที่เปลี่ยนไป ตามสมการ \(\vec{I} = \Delta \vec{p} = \vec{p}_f - \vec{p}_i\) ก่อนเบรกถือว่ามีโมเมนตัมเท่ากัน เมื่อเบรกจนหยุดโมเมนตัมเป็นศูนย์เหมือนกันทั้งสองกรณี ดังนั้นทั้งสองกรณีมีโมเมนตัมเปลี่ยนไปเท่ากัน

ข้อ 4 เป็นไปได้หรือไม่ ที่แรงดลที่มีค่ามากทำให้เกิดการดลที่มีค่าน้อยกว่าแรงดลที่มีค่าน้อย

คำตอบ : เป็นไปได้ เพราะการดล \(\vec{I} = \vec{F}\Delta t\) นั่นคือการดลยังขึ้นอยู่กับช่วงเวลาด้วย ถ้าแรงน้อยแต่มีช่วงเวลามากกวามากๆ สามารถทำให้เกิดการดลมากกว่าแรงมากแต่มีช่วงเวลาน้อยได้

รถยนต์กำลังแล่นไปตามถนน คนขับรถยนต์เห็นรถบรรทุกจอดนิ่งอยู่ข้างหน้าในระยะกระชั้นชิด เขาจึงเหยียบเบรกทันที ขณะที่ความเร็วของรถยนต์ลดลงเกือบหยุด รถยนต์ก็ชนรถบรรทุก ถ้ารถยนต์จะหยุดนิ่งภายในเวลา 5.0 \(\times\) 10-3 s แรงที่รถยนต์กระทำต่อรถบรรทุกเป็น 1.0 \(\times\) 106 N การดลที่กระทำต่อรถบรรทุกเป็นเท่าใด
