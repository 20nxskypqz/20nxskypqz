โมเมนตัม และการชน

6.5 การชนและการดีดตัวแยกจากกัน

1. การชน (Collisions)

เป็นการชนในหนึ่งมิติ (แนวเส้นตรง) แบ่งออกเป็น 2 ประเภทตามการคงอยู่ของพลังงานจลน์ :

การชนแบบยืดหยุ่น

  • โมเมนตัม : คงตัว (\(\sum \vec{P}_i = \sum \vec{P}_f\))
  • พลังงานจลน์ : คงตัว (\(\sum E_{ki} = \sum E_{kf}\))

ลักษณะ : ชนแล้วแยกจากกัน ไม่สูญเสียพลังงาน

การชนแบบไม่ยืดหยุ่น

  • โมเมนตัม : คงตัว (\(\sum \vec{P}_i = \sum \vec{P}_f\))
  • พลังงานจลน์ : ไม่คงตัว (ลดลง)

ลักษณะ : อาจบุบสลาย หรือชนแล้วติดกันไป

2. การดีดตัวแยกจากกัน

วัตถุเดิมติดอยู่ด้วยกันหรืออยู่นิ่ง แล้วแยกออกจากกันด้วยแรงภายใน

  • โมเมนตัม : คงตัว (ถ้าเดิมอยู่นิ่ง ผลรวมเป็นศูนย์)
  • พลังงานจลน์ : ไม่คงตัว (เพิ่มขึ้น) จากการเปลี่ยนรูปพลังงานศักย์

ตัวอย่าง : ยิงปืน, จรวด, สปริงดีดตัว

ประเภทเหตุการณ์ โมเมนตัม
(\(\vec{P}\))		 พลังงานจลน์
(\(E_k\))
การชนแบบยืดหยุ่น คงตัว คงตัว
การชนแบบไม่ยืดหยุ่น คงตัว ไม่คงตัว (ลดลง)
การระเบิด / ดีดตัว คงตัว ไม่คงตัว (เพิ่มขึ้น)

สมการที่ใช้คำนวณ

1. อนุรักษ์โมเมนตัม (ใช้ได้ทุกกรณี)
\(\sum \vec{P}_{\text{ก่อน}} = \sum \vec{P}_{\text{หลัง}}\)
\(m_1\vec{u}_1 + m_2\vec{u}_2 = m_1\vec{v}_1 + m_2\vec{v}_2\)
2. อนุรักษ์พลังงานจลน์ (เช็คยืดหยุ่น)
\(\sum E_{k\text{ก่อน}} = \sum E_{k\text{หลัง}}\)
\(\frac{1}{2}m_1u_1^2 + \frac{1}{2}m_2u_2^2 = \frac{1}{2}m_1v_1^2 + \frac{1}{2}m_2v_2^2\)

ตัวอย่างการคำนวณ

ตัวอย่าง 6.10

ยิงกระสุนปืนมวล 100 g ด้วยอัตราเร็ว 420 m/s เข้าหาแท่งไม้มวล 2.0 kg ที่วางอยู่นิ่งบนพื้นลื่นและฝังเข้าไปในแท่งไม้ ทำให้แท่งไม้เคลื่อนที่ไปด้วยอัตราเร็ว 20 m/s ระบบนี้มีการอนุรักษ์โมเมนตัมและการอนุรักษ์พลังงานจลน์หรือไม่

วิธีทำ

กำหนดให้ :
m1 (กระสุน) = 100 g = 0.1 kg
u1 = 420 m/s
m2 (ไม้) = 2.0 kg
u2 = 0 m/s
* หลังชน ติดกันไป (ฝัง) ความเร็วปลาย v = 20 m/s
1. ตรวจสอบการอนุรักษ์โมเมนตัม
\(\sum P_{\text{ก่อน}}\) = \(m_1 u_1 + m_2 u_2\) = \((0.1)(420) + 0 = 42 \text{ kg} \cdot \text{m/s}\) \(\sum P_{\text{หลัง}}\) = \((m_1 + m_2)v\) = \((0.1 + 2.0)(20) = (2.1)(20) = 42 \text{ kg} \cdot \text{m/s}\)

โมเมนตัมคงตัว (อนุรักษ์)

2. ตรวจสอบการอนุรักษ์พลังงานจลน์
\(\sum E_{k\text{ก่อน}}\) = \(\frac{1}{2}m_1 u_1^2 + 0\) = \(\frac{1}{2}(0.1)(420)^2 = 8,820 \text{ J}\) \(\sum E_{k\text{หลัง}}\) = \(\frac{1}{2}(m_1 + m_2)v^2\) = \(\frac{1}{2}(2.1)(20)^2 = 420 \text{ J}\)

พลังงานจลน์ไม่คงตัว

คำตอบ

มีการอนุรักษ์โมเมนตัม แต่ ไม่มีการอนุรักษ์พลังงานจลน์

ตัวอย่าง 6.11

กล่องมวล 150 g ให้เคลื่อนที่ไปทางขวาบนลานน้ำแข็งด้วยอัตราเร็ว 3.0 m/s ในขณะที่กล่องมวล 300 g ให้เคลื่อนที่ไปทางซ้ายบนลานน้ำแข็งด้วยอัตราเร็ว 3.0 m/s กล่องทั้งสองเคลื่อนที่อยู่ในแนวเดียวกัน และมาชนกัน หลังการชน กล่องมวล 300 g เคลื่อนที่ไปทางขวาด้วยอัตราเร็ว 1.0 m/s
ก. กล่องมวล 150 g เคลื่อนที่ไปในทิศทางใดและมีอัตราเร็วเท่าใด ถ้าไม่พิจารณาแรงเสียดทานจากพื้นน้ำแข็งซึ่งมีขนาดน้อยมาก
ข. การชนในสถานการณ์นี้เป็นการชนแบบยืดหยุ่นหรือไม่ เพราะเหตุใด

วิธีทำ

กำหนดให้ (ขวา + , ซ้าย -) :
m1 = 0.15 kg
u1 = +3.0 m/s
m2 = 0.30 kg
u2 = -3.0 m/s
v2 = +1.0 m/s
ก. หาความเร็วกล่อง 150 g (\(v_1\))
\(\sum P_{\text{ก่อน}}\) = \(\sum P_{\text{หลัง}}\) \(m_1 u_1 + m_2 u_2\) = \(m_1 v_1 + m_2 v_2\) \((0.15)(3) + (0.3)(-3)\) = \((0.15)v_1 + (0.3)(1)\) \(0.45 - 0.90\) = \(0.15v_1 + 0.30\) \(-0.45\) = \(0.15v_1 + 0.30\) \(-0.75\) = \(0.15v_1\) \(v_1\) = -5.0 \(\text{m/s}\)

เครื่องหมายลบ แสดงว่าไปทางซ้าย

ข. ตรวจสอบพลังงานจลน์
\(\sum E_{k\text{ก่อน}}\) = \(\frac{1}{2}(0.15)(3)^2 + \frac{1}{2}(0.3)(3)^2\) = \(0.675 + 1.35 = \mathbf{2.025 \text{ J}}\) \(\sum E_{k\text{หลัง}}\) = \(\frac{1}{2}(0.15)(5)^2 + \frac{1}{2}(0.3)(1)^2\) = \(1.875 + 0.15 = \mathbf{2.025 \text{ J}}\)

คำตอบ

ก. กล่อง 150 g เคลื่อนที่ไปทางซ้ายด้วยอัตราเร็ว 5.0 m/s

ข. เป็นการชนแบบ ยืดหยุ่น เพราะพลังงานจลน์ของระบบคงตัว

ตัวอย่าง 6.12

นักบินอวกาศใช้ชุดขับเคลื่อนสำรวจสภาพแวดล้อมภายนอกของยานอวกาศ ซึ่งมีสภาพที่ แรงลัพธ์ภายนอกที่กระทำต่อนักบินอวกาศน้อยมากจนถือว่าเป็นศูนย์
ขณะที่นักบินอวกาศอยู่ห่างจากยานอวกาศ 10 เมตร ชุดขับเคลื่อนของเขาขัดข้อง ทำให้ไม่สามารถ เคลื่อนที่กลับไปที่ยานอวกาศได้ เขาจึงใช้กฎการอนุรักษ์โมเมนตัมในการช่วยให้ตัวเขาเคลื่อนที่กลับไปยัง ยานอวกาศโดยถอดชุดขับเคลื่อนแล้วผลักชุดขับเคลื่อนที่มีมวล 140 kg ไปในทิศทางออกจากยาน อวกาศ ถ้านักบินอวกาศและชุดของเขามีมวลรวมกัน 120 kg เขาควรจะผลักชุดขับเคลื่อนออกไป ด้วยความเร็วเท่าใด เพื่อให้เขามีความเร็ว 0.7 m/s ไปในทิศทางกลับเข้าหายานอวกาศ

วิธีทำ

กำหนดให้ (ทิศเข้าหายาน +) :
m1 (คน+ชุด) = 120 kg
v1 = +0.7 m/s
m2 (ชุดขับเคลื่อน) = 140 kg
u (เดิมอยู่นิ่งรวมกัน) = 0
ขั้นตอนที่ 1 : กฎการอนุรักษ์โมเมนตัม
\(\sum P_{\text{ก่อน}}\) = \(\sum P_{\text{หลัง}}\) \(0\) = \(m_1 v_1 + m_2 v_2\) \(0\) = \((120)(0.7) + (140)v_2\) \(0\) = \(84 + 140v_2\) \(-84\) = \(140v_2\) \(v_2\) = -0.6 \(\text{m/s}\)

( เครื่องหมายลบ แสดงว่าต้องผลักไปในทิศตรงข้ามกับยาน )

คำตอบ

เขาควรผลักชุดขับเคลื่อนด้วยอัตราเร็ว 0.6 m/s

คำถามตรวจสอบความเข้าใจ 6.5

ข้อ 1 การชนแบบยืดหยุ่นและการชนแบบไม่ยืดหยุ่นเหมือนและแตกต่างกันอย่างไร

คำตอบ : การชนทั้งสองแบบมี โมเมนตัมคงตัว เหมือนกัน แต่แตกต่างกันที่ พลังงานจลน์ โดยแบบยืดหยุ่นพลังงานจลน์คงตัว ส่วนแบบไม่ยืดหยุ่นพลังงานจลน์ไม่คงตัว (ลดลง)

ข้อ 2 การชนแบบไม่ยืดหยุ่น พลังงานจลน์ของระบบคงตัวหรือไม่ เป็นเพราะเหตุใด

คำตอบ : ไม่คงตัว เพราะมีการเปลี่ยนพลังงานจลน์บางส่วนไปเป็นงานของแรงต้าน หรือพลังงานอื่น เช่น เสียง ความร้อน หรือการเปลี่ยนรูปร่าง

ข้อ 3 การชนกันของวัตถุแล้วติดกันไปเป็นการชนแบบยืดหยุ่นหรือการชนแบบไม่ยืดหยุ่น เพราะเหตุใด

คำตอบ : เป็นการชนแบบ ไม่ยืดหยุ่น (สมบูรณ์) เพราะมีการสูญเสียพลังงานจลน์ไปมากที่สุดในการทำให้วัตถุติดกันไป

ข้อ 4 ถ้าวัตถุมวลมากชนวัตถุมวลน้อยกว่าที่อยู่นิ่ง โมเมนตัมของวัตถุทั้งสองจะเปลี่ยนหรือไม่ อย่างไร

คำตอบ : โมเมนตัมของแต่ละวัตถุ เปลี่ยนแปลง แต่ผลรวมโมเมนตัมของระบบ ไม่เปลี่ยนแปลง (คงตัว) ตามกฎการอนุรักษ์โมเมนตัม
