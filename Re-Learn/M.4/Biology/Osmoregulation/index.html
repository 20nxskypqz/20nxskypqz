2.1 การรักษาดุลยภาพของน้ำและสารในร่างกาย

ระดับน้ำ
แร่ธาตุ

ของเสียและการขับถ่าย

ทำความเข้าใจสิ่งที่ร่างกายกำจัดออก

ของเสียที่ต้องกำจัด ได้แก่

  • ปัสสาวะ
  • เหงื่อ
  • อุจจาระ
  • แก๊สคาร์บอนไดออกไซด์
  • ยูเรีย

เมแทบอลิซึม (Metabolism)

คือ ปฏิกิริยาเคมีต่าง ๆ ที่เกิดขึ้นภายในสิ่งมีชีวิต เช่น การหายใจระดับเซลล์เพื่อให้ได้พลังงานสำหรับการดำรงชีวิต

ความเข้าใจเรื่อง
"การถ่ายอุจจาระ" vs "การขับถ่าย"

ความเข้าใจคลาดเคลื่อน

ของเสียที่เกิดจากการทำงานของระบบขับถ่าย (excretory system) ของร่างกายมนุษย์จะถูกกำจัดออกไปอยู่ในรูปของปัสสาวะ และอุจจาระ

ความเข้าใจที่ถูกต้อง

การขับถ่าย หมายถึง การกำจัดของเสียซึ่งเกิดจากกระบวนการเมแทบอลิซึมในเซลล์ซึ่งมีไนโตรเจนเป็นองค์ประกอบ และจะถูกขับถ่ายออกในรูปของปัสสาวะ

การถ่ายอุจจาระ (defecation) ไม่จัดเป็นการขับถ่าย เพราะอุจจาระเป็นกากอาหารที่ร่างกายย่อยไม่ได้ในระบบย่อยอาหาร (digestive system)

สำหรับของเสียอื่น ๆ เช่น เหงื่อถูกกำจัดออกทางผิวหนัง แก๊สคาร์บอนไดออกไซด์ถูกกำจัดออกโดยระบบหายใจ

สมดุลน้ำในร่างกาย

น้ำคือส่วนประกอบหลัก 65-70% ของน้ำหนักตัว

ดุลยภาพของน้ำ (Input vs Output)

รับเข้า (2,500 ml)
เครื่องดื่ม 1,600 ml
อาหาร 700 ml
เมแทบอลิซึม 200 ml
ขับออก (2,500 ml)
ปัสสาวะ 1,500 ml
การระเหย 700 ml
อุจจาระ 200 ml
เหงื่อ 100 ml

ร่างกายต้องรักษาปริมาณน้ำที่รับเข้าและขับออกให้สมดุลกัน เพื่อรักษาดุลยภาพ

การกระจายตัวของน้ำในร่างกาย

น้ำภายในเซลล์ (Intracellular fluid : ICF) 65%
น้ำภายนอกเซลล์ (Extracellular fluid : ECF) 35%
น้ำภายนอกเซลล์แบ่งย่อยเป็น 3 ส่วน ได้แก่
  1. น้ำภายในหลอดเลือด และน้ำเหลือง (Intravascular fluid)
  2. น้ำในเนื้อเยื่อระหว่างเซลล์ (Interstitial fluid)
  3. น้ำในช่องว่างต่างๆ (Transcellular fluid) เช่น น้ำในลูกตา เยื่อบุช่องท้อง เยื่อหุ้มปอด ไขข้อ น้ำในสมองและไขสันหลัง (CSF)

หน้าที่ของน้ำ

  • ตัวกลางในการเกิดปฏิกิริยาเคมี
  • ช่วยปรับอุณหภูมิร่างกาย
  • ช่วยลำเลียงแก๊ส สารอาหาร และของเสีย

ปริมาณการรับและเสียน้ำสูงสุด

  • ได้รับมากสุด (เครื่องดื่ม) 1,600 ml
  • สูญเสียมากสุด (ปัสสาวะ) 1,500 ml

กลไกการควบคุมสมดุลน้ำ

ร่างกายรู้ได้อย่างไร ?

เมื่อน้ำลดลงจะเกิดการ กระหายน้ำ และเมื่อน้ำมากจะ ปัสสาวะ ออกไป โดยมี สมองส่วนไฮโพทาลามัส (Hypothalamus) ทำหน้าที่ควบคุม

กรณี ร่างกายมีน้ำมาก / น้ำในเลือดเพิ่มขึ้น

ความเข้มข้นเลือดลดลง → ลดการกระตุ้นไฮโพทาลามัส → ต่อมใต้สมองส่วนหลังหลั่ง ADH ลดลง → ท่อหน่วยไตลดการดูดกลับน้ำ → ขับปัสสาวะปริมาณมากและเจือจาง

จะเกิดขึ้นชัดเจนในนาทีที่ 60-90 จนกลับสู่ปกติ

ระบบที่ประสานงานกัน ได้แก่ ระบบขับถ่าย ระบบหมุนเวียนเลือด ระบบประสาท และระบบต่อมไร้ท่อ (ฮอร์โมน)
ไดยูเรติก (Diuretic)

คือสารที่มีสมบัติทำให้เซลล์ท่อหน่วยไตสูญเสียน้ำออกไปมากกว่าการดูดกลับน้ำเข้ามา (ทำให้ปัสสาวะบ่อย) ตัวอย่างเช่น คาเฟอีน (ชา กาแฟ น้ำอัดลม ช็อกโกแลต)

1

การกรอง

(Filtration)

✓ ผ่านได้ (ขนาดเล็ก) น้ำ ยูเรีย กลูโคส กรดแอมิโน แร่ธาตุ
✗ ผ่านไม่ได้ (ขนาดใหญ่) เซลล์เม็ดเลือด เกล็ดเลือด โปรตีน
2

การดูดกลับ

(Reabsorption)

ท่อหน่วยไต & ท่อรวม

ดูดกลับหมด (มีประโยชน์) กลูโคส กรดแอมิโน

3

การหลั่ง

(Secretion)

ท่อหน่วยไต

หลั่งออกจากเลือดเข้าสู่ท่อไต

  • ไฮโดรเจนไอออน
  • แอมโมเนียมไอออน
  • สารพิษ
  • ยาบางชนิด
ข้อควรระวัง การทำงานของไตทั้ง 3 ขั้นตอนเกิดขึ้นต่อเนื่องตลอดเวลา (กรองเลือดวันละ ~180 ลิตร) หากไตบกพร่อง ของเสียจะสะสมจนเสียชีวิตได้

โรคและอาการผิดปกติ

เมื่อสมดุลร่างกายเปลี่ยนไป

โรคลมแดด (Heat Stroke)

มักเกิดในฤดูร้อน กับผู้สูงวัย เด็ก หรือผู้ที่ทำกิจกรรมกลางแจ้ง อุณหภูมิร่างกายสูงกว่า 40°C

อาการ ไม่มีเหงื่อ ปวดศีรษะ ทรงตัวไม่ดี หายใจเร็ว หัวใจเต้นเร็ว อาจช็อก/หมดสติ
การป้องกัน ดื่มน้ำบ่อยๆ เลี่ยงแดดจัด ใส่เสื้อผ้าสบายๆ อยู่ในที่เย็นๆ

การตรวจสารเสพติดในปัสสาวะ

สารเสพติด (ยาบ้า กัญชา ยาอี มอร์ฟิน เฮโรอีน) เมื่อผ่านไตและ ไม่ถูกดูดกลับ จะมีความเข้มข้นสูงในปัสสาวะ

สามารถใช้ชุดตรวจสำเร็จรูป (Screening test) คัดกรองเบื้องต้นได้ รวมถึงการตรวจ HCG (การตั้งครรภ์) และกลูโคส

ภาวะน้ำเป็นพิษ (Water Intoxication)

เกิดจากดื่มน้ำมากเกินไป (> 6-7 ลิตร/วัน)

  • เกิดภาวะโซเดียมในเลือดต่ำ (Hyponatremia)
  • น้ำแพร่เข้าเซลล์ ทำให้ เซลล์บวมน้ำ
  • อาการ ปวดศีรษะ คลื่นไส้ อาเจียน ชัก (สมองบวม) ปอดบวม เสียชีวิต

โรคเบาจืด (Diabetes Insipidus)

อาการ ปัสสาวะบ่อยและมาก กระหายน้ำตลอดเวลา (เสี่ยงภาวะขาดน้ำ/ช็อก)

สาเหตุ ร่างกายขาดฮอร์โมน ADH
  • ความผิดปกติของต่อมใต้สมองส่วนหลัง (เช่น เนื้องอก) ทำให้ไม่หลั่ง ADH
  • ไตผิดปกติ ไม่ตอบสนองต่อ ADH (เป็นแต่กำเนิด หรือเกิดจากยา)

ADH ปกติจะกระตุ้นให้ท่อขดส่วนท้ายและท่อรวมดูดน้ำกลับ

ผลกระทบเมื่อน้ำไม่สมดุล

หากร่างกายได้รับน้ำมากหรือน้อยเกินไป จะทำให้ความเข้มข้นเลือดเปลี่ยน ปฏิกิริยาเคมีไม่เกิด อันตรายต่อเซลล์ ระบบหมุนเวียนเลือดและระบบประสาท จนอาจเสียชีวิตได้

ดุลยภาพของแร่ธาตุ (โซเดียม)

แร่ธาตุสำคัญต่อระบบประสาทและกล้ามเนื้อ

ฮอร์โมนแอลโดสเทอโรน (Aldosterone)

แหล่งสร้าง ต่อมหมวกไตส่วนนอก (Adrenal cortex)

กลุ่ม มิเนราโลคอร์ติคอยด์

หน้าที่ ควบคุมปริมาณโซเดียม กระตุ้นให้ไตดูดกลับโซเดียมและน้ำเข้าหลอดเลือด

ถ้าขาดแอลโดสเทอโรน
สูญเสียโซเดียมและน้ำไปกับปัสสาวะ → ปริมาตรเลือดลด → ความดันต่ำ → เสียชีวิต
ภาวะเครียด (Long-term stress)
หลั่งมาก → ปริมาตรและความดันเลือดสูง

โซเดียมกับการบริโภค

หน้าที่ คุมสมดุลน้ำ รักษากรด-ด่าง

ปริมาณแนะนำ ไม่เกิน 2,000 มก./วัน (เกลือ 1 ช้อนชา)

(หากออกกำลังกายเสียเหงื่อมาก ต้องการมากขึ้น)
คำเตือน คนไทยได้รับโซเดียมเกิน 2 เท่า ! (จากเครื่องปรุง) เสี่ยงโรคความดันโลหิตสูง ส่งผลให้ไตทำงานผิดปกติ (พบโปรตีนในปัสสาวะ) เกิดอาการบวม แน่นหน้าอก

แผนผังการควบคุมสมดุลโซเดียมและน้ำ (แอลโดสเทอโรน)
