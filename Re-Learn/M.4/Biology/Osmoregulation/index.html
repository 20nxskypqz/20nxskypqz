2.1 การรักษาดุลยภาพของน้ำและสารในร่างกาย

ของเสียที่เกิดขึ้นในร่างกายที่ต้องถูกกำจัดออกจากร่างกายได้แก่อะไรบ้าง ?

ปัสสาวะ เหงื่อ อุจจาระ แก๊สคาร์บอนไดออกไซด์ ยูเรีย

การถ่ายอุจจาระ (defecation) จัดเป็นการขับถ่าย (excretion) หรือไม่ เพราะเหตุใด ?

การถ่ายอุจจาระไม่เป็นการขับถ่าย เพราะการขับถ่าย หมายถึง การกำจัดของเสียซึ่งเกิดจาก กระบวนการเมแทบอลิซึม ในเซลล์ซึ่งมีไนโตรเจนเป็นองค์ประกอบ แต่การถ่ายอุจจาระเป็นการ กำจัดกากอาหารที่ร่างกายย่อยไม่ได้ออกจากร่างกาย

ความเข้าใจคลาดเคลื่อน
ความเข้าใจที่ถูกต้อง
ของเสียที่เกิดจากการทำงานของระบบ ขับถ่าย (excretory system) ของร่างกาย มนุษย์จะถูกกำจัดออกไปอยู่ในรูปของปัสสาวะ และอุจจาระ

ระบบขับถ่ายของร่างกายมนุษย์เป็นการ กำจัดของเสียที่เกิดจากกระบวนการเมแทบอลิซึม ที่มีไนโตรเจนเป็นองค์ประกอบ และจะถูกขับถ่ายออกในรูปของ ปัสสาวะ

ส่วน อุจจาระ เป็นกากอาหารที่ร่างกายย่อย ไม่ได้ในระบบย่อยอาหาร (digestive system)

สำหรับของเสียอื่น ๆ เช่น เหงื่อ ถูกกำจัดออก ทางผิวหนัง แก๊สคาร์บอนไดออกไซด์ ถูกกำจัด ออกโดยระบบหายใจ

โรคลมแดดและการรักษาดุลยภาพของน้ำ

ฮีทสโตรก (Heat Stroke)

ในฤดูร้อนของประเทศไทย มักจะพบผู้เป็นโรคลมแดด หรือฮีทสโตรกอยู่เสมอ โดยจะเกิดกับผู้ที่ มีสุขภาพอ่อนแอ ผู้สูงวัย เด็กเล็ก และผู้ที่ประกอบกิจกรรม กลางแจ้ง ผู้ป่วยจะมีอุณหภูมิภายในร่างกายสูงกว่า 40 องศาเซลเซียส ไม่มีเหงื่อ ปวดศีรษะหรือมึนศีรษะ การทรง ตัวไม่ดี หายใจเร็วและตื้น หัวใจเต้นเร็ว เหนื่อย ถ้าอาการ หนักจะช็อกและหมดสติ อาจทำให้เสียชีวิตได้

การป้องกัน :

  • ดื่มน้ำบ่อยๆ เพื่อชดเชยการเสียเหงื่อ
  • เลี่ยงแดดจัด หรืออยู่ในที่อากาศถ่ายเทสะดวก
  • ใส่เสื้อผ้าสบายๆ ระบายความร้อนได้ดี

น้ำในร่างกาย

ร่างกายมนุษย์มีน้ำประมาณร้อยละ 65-70 ของน้ำหนักตัว พบได้ทั้งภายในเซลล์และภายนอกเซลล์ น้ำทำหน้าที่เป็น ตัวกลางในการเกิดปฏิกิริยาเคมี ช่วยปรับอุณหภูมิของร่างกาย ช่วยลำเลียงแก๊ส สารอาหารและของเสีย ในแต่ละวันมนุษย์ ต้องดื่มน้ำให้เพียงพอต่อความต้องการของร่างกาย

การสูญเสียน้ำขึ้นกับอุณหภูมิและ ความชื้นในอากาศ เช่น เมื่ออยู่กลางแดด ร่างกายจะขับเหงื่อ ออกมากกว่าปกติซึ่งเหงื่อที่ขับออกมาจะมีทั้งน้ำและแร่ธาตุปน ออกมาด้วย หรือเมื่ออุณหภูมิภายนอกต่ำลง ร่างกายจะขับ ปัสสาวะบ่อยกว่าปกติ

การรักษาดุลยภาพ (homeostasis) คือ การรักษา สภาพแวดล้อมภายในสิ่งมีชีวิตให้สมดุล และเหมาะสมต่อการดำรงชีวิต

การกระจายของน้ำในร่างกาย

  • น้ำภายในเซลล์ (Intracellular fluid : ICF) : 65%
  • น้ำภายนอกเซลล์ (Extracellular fluid : ECF) : 35%
    • 1. น้ำในหลอดเลือดและน้ำเหลือง (Intravascular fluid)
    • 2. น้ำในเนื้อเยื่อระหว่างเซลล์ (Interstitial fluid)
    • 3. น้ำในช่องว่างต่างๆ (Transcellular fluid) เช่น น้ำไขข้อ น้ำในสมองและไขสันหลัง (CSF)

ความสำคัญของสมดุลน้ำ

ถ้าปริมาณน้ำที่ร่างกายได้รับและสูญเสียไม่สมดุลกัน จะทำให้ความเข้มข้นของเลือดเปลี่ยนแปลง การลำเลียงสารต่างๆ หรือการเกิดปฏิกิริยาเคมีจะไม่เกิดขึ้น ซึ่งเป็นอันตรายต่อเซลล์และระบบต่างๆ ของร่างกายจนอาจเสียชีวิตได้

* เมแทบอลิซึม (metabolism) คือ ปฏิกิริยาเคมีต่างๆ ที่เกิดขึ้นภายในสิ่งมีชีวิต เช่น การหายใจระดับเซลล์เพื่อให้ได้พลังงาน

การควบคุมน้ำ (ADH)

สมองส่วน ไฮโพทาลามัส ควบคุมการหลั่งฮอร์โมน ADH (Antidiuretic hormone) จากต่อมใต้สมองส่วนหลัง เพื่อกระตุ้นให้ท่อขดส่วนท้ายและท่อรวมดูดน้ำกลับเข้าหลอดเลือด

  • น้ำมาก : ADH หลั่งน้อย -> ดูดกลับน้อย -> ปัสสาวะมาก/เจือจาง
  • น้ำน้อย : ADH หลั่งมาก -> ดูดกลับมาก -> ปัสสาวะน้อย/เข้มข้น
  • ไดเยูเรติก (Diuretic) : สารขับปัสสาวะ เช่น คาเฟอีน (ชา กาแฟ น้ำอัดลม) ยับยั้งการดูดน้ำกลับ ทำให้ปัสสาวะบ่อย

โรคและความผิดปกติเกี่ยวกับน้ำ

ภาวะน้ำเป็นพิษ (Water Intoxication) ดื่มน้ำมากเกินไป (>6-7 ลิตร/วัน) -> โซเดียมในเลือดต่ำ (Hyponatremia) -> เซลล์บวมน้ำ (โดยเฉพาะสมอง) -> ปวดหัว ชัก เสียชีวิต
โรคเบาจืด (Diabetes Insipidus) ขาด ADH หรือไตไม่ตอบสนองต่อ ADH -> ดูดน้ำกลับไม่ได้ -> ปัสสาวะบ่อยและมาก (เจือจาง) -> กระหายน้ำตลอดเวลา -> เสี่ยงภาวะขาดน้ำ (Dehydration) ช็อก

การกำจัดของเสียและรักษาดุลยภาพของสาร

อวัยวะในระบบขับถ่ายของมนุษย์

  • ไต (Kidney) : กำจัดของเสียไนโตรเจน
  • ผิวหนัง : ขับเหงื่อ
  • ปอด : ขับ CO₂
  • ลำไส้ใหญ่ : ขับกากอาหาร (ไม่นับเป็น Excretion)

โครงสร้างระบบปัสสาวะ

  • ไต 2 ข้าง (Kidneys)
  • ท่อไต (Ureter) : ลำเลียงปัสสาวะไปกระเพาะปัสสาวะ
  • กระเพาะปัสสาวะ (Urinary Bladder) : สะสมปัสสาวะ
  • ท่อปัสสาวะ (Urethra) : ขับปัสสาวะออกนอกร่างกาย

หน่วยไต (Nephron)

ไตแต่ละข้างมีหน่วยไตประมาณ 1 ล้านหน่วย

กระบวนการทำงานของไต 3 ขั้นตอน

1. การกรอง (Filtration)

เกิดขึ้นที่ โกลเมอรูลัส และ โบว์แมนส์แคปซูล
• สารขนาดเล็กผ่านได้ : น้ำ กลูโคส ยูเรีย กรดแอมิโน แร่ธาตุ
• สารขนาดใหญ่ผ่านไม่ได้ : เม็ดเลือด เกล็ดเลือด โปรตีน

2. การดูดกลับ (Reabsorption)

เกิดขึ้นที่ ท่อหน่วยไต
• สารมีประโยชน์ถูกดูดกลับ : กลูโคส กรดแอมิโน (ดูดกลับหมด)
• ดูดกลับบางส่วน : น้ำ โซเดียม โพแทสเซียม ยูเรียบางส่วน

3. การหลั่ง (Secretion)

เกิดขึ้นที่ ท่อหน่วยไต
• ขับสารส่วนเกิน/พิษออกจากเลือดเข้าสู่ท่อไต : ไฮโดรเจนไอออน แอมโมเนียมไอออน ยาบางชนิด สารพิษ

เกร็ดความรู้ : การตรวจสารเสพติดและโรคไต

สารเสพติดบางชนิด เช่น ยาบ้า (methamphetamine) กัญชา ยาอี มอร์ฟิน เฮโรอีน เมื่อผ่านไตและไม่ถูกดูดกลับจะมีความเข้มข้นสูงในปัสสาวะ จึงใช้ ตรวจคัดกรองเบื้องต้น (Screening test) ได้

ในแต่ละวันไตต้องกรองสารจากเลือดมากถึง 180 ลิตร หากไตบกพร่องจะทำให้ของเสียสะสมในเลือด รักษาดุลยภาพไม่ได้ และอาจเสียชีวิต

การควบคุมแร่ธาตุ (Aldosterone)

แอลโดสเตอโรน สร้างจากต่อมหมวกไตส่วนนอก ควบคุมสมดุล โซเดียม และโพแทสเซียม

  • กระตุ้นให้ไต ดูดกลับโซเดียมและน้ำ เข้าหลอดเลือด
  • ขับโพแทสเซียมออกจากเลือด
  • ถ้าขาดฮอร์โมนนี้ : ร่างกายสูญเสียโซเดียมและน้ำ -> ปริมาตรเลือดลด -> ความดันต่ำ -> เสียชีวิตได้
  • หลั่งมากเมื่อเครียด (Long-term stress) -> ความดันสูง

โซเดียมกับคนไทย

คนไทยได้รับโซเดียมมากถึง 2 เท่า ของปริมาณที่แนะนำ (ไม่เกิน 2000 มก./วัน หรือเกลือ 1 ช้อนชา) จากเครื่องปรุงรส ทำให้เสี่ยงโรคความดันโลหิตสูงและโรคไต หากไตขับโซเดียมไม่ได้ จะเกิดอาการบวมน้ำ แน่นหน้าอก
