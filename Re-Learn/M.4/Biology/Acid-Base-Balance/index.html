ส่วนประกอบของเลือด

ระบบไหลเวียนที่สำคัญทั่วร่างกาย

น้ำเลือด / พลาสมา

  • เป็นของเหลว
  • ประกอบด้วย น้ำ สารอาหาร เอนไซม์ ฮอร์โมน ของเสีย
  • ลำเลียงสารไปทั่วร่างกาย

เซลล์เม็ดเลือดแดง (Erythrocyte)

  • กลมแบน ตรงกลางบุ๋ม ไม่มีนิวเคลียส
  • มีเฮโมโกลบิน
  • ลำเลียงแก๊สออกซิเจน

เซลล์เม็ดเลือดขาว (Leucocyte)

  • ใหญ่กว่าเม็ดเลือดแดง มีนิวเคลียส
  • สร้างแอนติบอดี / กลืนกินเซลล์
  • ทำลายเชื้อโรคและสิ่งแปลกปลอม

กลไกการรักษาดุลยภาพกรด-เบส

สมการการเกิดกรดคาร์บอนิกในเลือด

ความสัมพันธ์ : ความเข้มข้นของ H⁺ เปลี่ยนแปลงตามกิจกรรมต่างๆ

H⁺ เพิ่ม = เลือดเป็นกรด H⁺ ลด = เลือดเป็นเบส

1. การหายใจ (ปอด)

ปอดแลกเปลี่ยนแก๊สบริเวณ ถุงลมปอด เพื่อกำจัด CO₂ ที่เกิดจากเมแทบอลิซึม

ภาวะเลือดเป็นกรด (CO₂ / H⁺ สูง)

กระตุ้นสมองส่วนควบคุมการหายใจ → หายใจเร็วและลึก → ขับ CO₂ ออกเร็วขึ้น

ภาวะเลือดเป็นเบส

อัตราการหายใจลดลง → สะสม CO₂ ในเลือดเพิ่มขึ้น → เพิ่ม H⁺

* เรากลั้นหายใจไม่ได้นานเพราะ CO₂ สะสมจนร่างกายทนไม่ได้ ต้องบังคับให้หายใจออก (อยู่นอกอำนาจจิตใจ)

2. การทำงานของไต

การตอบสนองของท่อหน่วยไต

เมื่อเลือดเป็นกรด
หลั่ง H⁺ NH₄⁺ เข้าสู่ปัสสาวะ
ดูดกลับ HCO₃⁻ Na⁺ เข้าสู่เลือด
เมื่อเลือดเป็นเบส
ลดการหลั่ง H⁺
ลดการดูดกลับสารที่เป็นเบส (HCO₃⁻)

* กำจัดไอออนส่วนเกินโดยกระบวนการ หลั่ง (Secretion)

3. ระบบบัฟเฟอร์ (Buffer System)

คือ สารละลายที่ต้านทานการเปลี่ยนแปลง pH ให้เปลี่ยนไปน้อยที่สุด

  • ประกอบด้วย กรดอ่อน+เกลือของกรดอ่อน หรือ เบสอ่อน+เกลือของเบสอ่อน
  • ตัวอย่าง : ไฮโดรเจนคาร์บอเนตไอออน ฟอสเฟตไอออน โปรตีน
  • ข้อดี : ปรับ pH ได้อย่างรวดเร็ว

โรคเกี่ยวกับไตและทางเดินปัสสาวะ

สาเหตุ อาการ และการป้องกัน

สาเหตุและความเสี่ยง

  • พันธุกรรม
  • โรคประจำตัว (เบาหวาน ความดันสูง เก๊าท์)
  • ทานอาหารรสจัด (เค็ม หวาน เผ็ด)
  • ดื่มน้ำน้อย ไม่ออกกำลังกาย เครียด
  • อายุ > 60 ปี

อาการเตือน

  • อ่อนเพลีย ไม่มีแรง
  • ปัสสาวะผิดปกติ (สี/กลิ่น)
  • ตัวบวม (น้ำ/เกลือเกิน)
  • ปวดหลัง ปวดบั้นเอว
  • คลื่นไส้ อาเจียน

* ระยะแรกมักไม่แสดงอาการ

1. โรคไตวาย (Renal Failure)

ภาวะที่ไตสูญเสียหน้าที่ สะสมของเสียในร่างกาย

• เฉียบพลัน : เกิดเร็ว (อุบัติเหตุ สารพิษ นิ่วอุดตัน) -> อาจฟื้นฟูได้

• เรื้อรัง : ค่อยเป็นค่อยไป (เบาหวาน ความดัน) -> รักษาไม่หาย ต้องฟอก/ปลูกถ่ายไต

2. โรคนิ่วในไต (Kidney Stone)

ตะกอน (เช่น แคลเซียม ฟอสเฟต) จับตัวเป็นก้อนแข็งอุดตัน

เสี่ยง : ทานผักที่มีออกซาเลตสูง (ผักโขม หน่อไม้ ชะพลู)

3. การอักเสบติดเชื้อ

  • กระเพาะปัสสาวะอักเสบ (Cystitis) : เชื้อ E. coli จากลำไส้ (หญิงเป็นง่ายกว่าชาย)
  • ท่อปัสสาวะอักเสบ (Urethritis) : เชื้อจากเพศสัมพันธ์ หรือการระคายเคือง

การบำบัดทดแทนไต (สำหรับไตวายเรื้อรัง)

การฟอกเลือด เครื่องฟอกไต (4-5 ชม./ครั้ง)
ฟอกทางช่องท้อง ทำเองได้ (4 รอบ/วัน)
ปลูกถ่ายไต รับไตจากผู้บริจาค

การตรวจสุขภาพ (การทำงานของไต)

การตรวจปัสสาวะ

  • ความเป็นกรด-เบส (pH)
  • ความถ่วงจำเพาะ
  • โปรตีน และ กลูโคส
  • ตะกอน (เม็ดเลือด แบคทีเรีย ผลึก)

การตรวจเลือด

  • BUN (Blood Urea Nitrogen) ค่ายูเรียไนโตรเจน
  • Creatinine (ครีเอทินีน) สารจากกล้ามเนื้อ
