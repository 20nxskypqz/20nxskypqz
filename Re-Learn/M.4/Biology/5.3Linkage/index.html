"ยีนที่อยู่บนโครโมโซมเดียวกัน ... จะถูกถ่ายทอดไปด้วยกันหรือไม่ ?"

ลิงเกจ (Linkage)

คือ กลุ่มของยีนที่มีตำแหน่งอยู่บน โครโมโซมแท่งเดียวกัน ซึ่งมักจะถูกถ่ายทอดไปพร้อมๆ กันสู่รุ่นลูกหลาน (โดยเฉพาะถ้ายีนอยู่ใกล้ชิดกันมาก)

กรณีศึกษา : แมลงหวี่ของมอร์แกน

กำหนดสัญลักษณ์พันธุกรรม :

1. ลักษณะสีลำตัว
b+ = น้ำตาล (เด่น) b = ดำ (ด้อย)
2. ลักษณะปีก
vg+ = ปกติ (เด่น) vg = กุด (ด้อย)

เปรียบเทียบรูปแบบการถ่ายทอด 3 กรณี

กรณี 1 : กฎของเมนเดล (แยกอิสระ)

พบ ลักษณะใหม่ (Recombinant) จำนวนน้อย เกิดจากการแลกเปลี่ยนชิ้นส่วน

คำอธิบาย : ครอสซิงโอเวอร์ (Crossing Over)

เกิดในระยะ Prophase I ของการแบ่งเซลล์แบบไมโอซิส เมื่อโครโมโซมคู่เหมือนมาเข้าคู่กัน และเกิดการไขว้กันของโครมาทิด

  • Parental Type : แบบเดิม (ไม่ไขว้) พบจำนวนมาก
  • Recombinant Type : แบบใหม่ (เกิดจากการไขว้) พบจำนวนน้อย
