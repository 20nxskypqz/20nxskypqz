Kidbright Smart Farm

1 แนวคิดของระบบ Smart Farm

โปรเจกต์นี้จำลองสถานการณ์ของ "เจมส์" เจ้าของฟาร์มเนื้อที่ 1,000 ไร่ ที่ต้องการนำเทคโนโลยีมาใช้ควมคุมการรดน้ำต้นไม้แทนคนงาน เพื่อวัตถุประสงค์หลักคือ :

  • เพิ่มผลผลิต ให้กับฟาร์ม
  • ลดต้นทุน ในการจ้างแรงงาน
  • ทำงานอัตโนมัติ : ระบบจะรดน้ำเมื่อดินแห้ง และหยุดเมื่อดินมีความชื้น

2 อุปกรณ์ที่ใช้ในระบบ

ระบบแบ่งออกเป็น 3 ส่วนหลักตามโครงสร้างคอมพิวเตอร์พื้นฐาน :

ส่วนการทำงาน อุปกรณ์ที่ใช้
Input ( ตัวป้อน ) เซนเซอร์วัดความชื้นในดิน
Process ( การประมวลผล ) บอร์ด KidBright
Output ( ตัวส่งออก ) ปั๊มน้ำขนาดเล็ก

3 การต่อวงจร ( Wiring )

เพื่อให้ระบบทำงานได้ ต้องต่อสายสัญญาณให้ตรงกับตำแหน่งในโปรแกรม ดังนี้ :

การต่อเซนเซอร์วัดความชื้น

  • DO ต่อเข้าช่อง IN 1
  • GND ต่อเข้าช่อง GN
  • VCC ต่อเข้าช่อง 5V

การต่อปั๊มน้ำขนาดเล็ก

  • สายสีดำ ( GND ) ต่อเข้าช่อง OUT 1
  • สายสีแดง ( VCC ) ต่อเข้าช่อง 3V3
