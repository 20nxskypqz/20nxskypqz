Kidbright IoT

1 IoT คืออะไร ?

  • IoT ย่อมาจาก Internet of Things
  • เกิดจากการรวมคำว่า Things ( สิ่งของหลากหลายประเภท ) กับ Internet ( เครือข่ายเชื่อมต่ออุปกรณ์อิเล็กทรอนิกส์ )
  • ความหมาย : คือการที่อุปกรณ์ต่าง ๆ รอบตัวเราสามารถเชื่อมต่อและสื่อสารกันได้ผ่านระบบอินเทอร์เน็ต

2 หลักการทำงานและอุปกรณ์

  • กระบวนการ : เริ่มจาก โทรศัพท์ ส่งสัญญาณผ่าน อินเทอร์เน็ต ( Cloud ) ไปยัง ไมโครคอนโทรลเลอร์ ( KidBright ) เพื่อควบคุมอุปกรณ์ต่าง ๆ เช่น หลอดไฟ
  • แอปพลิเคชัน : ต้องติดตั้งแอปฯ ชื่อ "KidBright IoT" ( พัฒนาโดย NEXPIE ) ลงในมือถือ

  • การตั้งค่า Wi-Fi : ต้องระบุชื่อ Wi-Fi ( SSID ) และรหัสผ่านให้ถูกต้อง ( ตัวพิมพ์เล็ก-ใหญ่และเว้นวรรคต้องแม่นยำ ) เพื่อให้บอร์ดเชื่อมต่ออินเทอร์เน็ตได้
  • การ Sync ข้อมูล : เมื่ออัปโหลดโปรแกรมลงบอร์ดแล้ว ให้ใช้แอปฯ ในมือถือสแกน QR Code จากหน้าจอโปรแกรม KidBright เพื่อเชื่อมโยงอุปกรณ์
  • การตรวจสอบสถานะ : หากเชื่อมต่อสำเร็จ ไฟสถานะ WIFI และ IOT บนบอร์ด KidBright จะสว่างขึ้นทั้ง 2 จุด ( 1.WIFI 2.IoT )
