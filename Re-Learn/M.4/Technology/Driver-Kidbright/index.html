2.1

วิธีติดตั้งไดรฟ์เวอร์คิดไบร์ท ( Driver Kidbright )

ภาพรวม

สรุปขั้นตอนการติดตั้ง Driver KidBright เพื่อให้คอมพิวเตอร์สามารถสื่อสารกับบอร์ด KidBright ได้อย่างถูกต้อง

ขั้นตอนที่ 1 : การดาวน์โหลด Driver

  • เข้าสู่เว็บไซต์ Google แล้วค้นหาคำว่า "kidbright download"
  • เลือกผลลัพธ์จากเว็บไซต์ www.kid-bright.org ในหัวข้อ "KidBright Download - Coding at School"
  • เมื่อเข้าสู่หน้าดาวน์โหลด ให้เลื่อนไปที่ส่วนของ All Versions ( Windows )
  • คลิกดาวน์โหลดที่หัวข้อ "FTDI Driver ( สำหรับ KidBright32 version 1.3 )"

ขั้นตอนที่ 2 : การแตกไฟล์ ( Extract File )

  • หลังจากดาวน์โหลดเสร็จสิ้น ให้เปิดเข้าไปที่ This PC บนหน้าจอคอมพิวเตอร์ของคุณ
  • เลือกไปที่โฟลเดอร์ Downloads
  • หาไฟล์ชื่อ CDM21228_Setup ( ซึ่งจะเป็นไฟล์บีบอัดหรือ Zip File ) จากนั้นให้ คลิกขวา
  • เลือกเมนู "แยกไฟล์ลงที่นี่" ( Extract Here )
  • คุณจะได้โฟลเดอร์ใหม่ที่ชื่อว่า CDM21228_Setup ปรากฏขึ้นมา
3

ขั้นตอนที่ 3 : การเริ่มติดตั้ง ( Installation )

เมื่อเตรียมไฟล์เรียบร้อยแล้ว เข้าสู่กระบวนการ ติดตั้งไดรฟ์เวอร์ ลงสู่ระบบปฏิบัติการ ประกอบด้วยขั้นตอนสำคัญดังนี้ :

เปิดโฟลเดอร์
Extract Driver
Finish

3 รายละเอียดการติดตั้ง

เริ่มรันโปรแกรม

  • เปิดเข้าไปในโฟลเดอร์ที่เพิ่งแตกไฟล์ออกมา
  • แล้วทำการ ดับเบิลคลิก ที่ไอคอนโปรแกรม CDM21228_Setup

ข้อตกลงและติดตั้ง

  • เมื่อหน้าต่าง FTDI CDM Drivers ปรากฏขึ้น ให้กดปุ่ม Extract
  • เข้าสู่หน้า License Agreement ( ข้อตกลงการใช้งาน ) : - เลือกช่อง "I accept this agreement" - จากนั้นกดปุ่ม Next >

เสร็จสิ้นสมบูรณ์

  • เมื่อการติดตั้งเสร็จสมบูรณ์ จะขึ้นหน้าต่าง Completing the Device Driver Installation Wizard
  • กดปุ่ม Finish เพื่อเสร็จสิ้นขั้นตอนการติดตั้ง
