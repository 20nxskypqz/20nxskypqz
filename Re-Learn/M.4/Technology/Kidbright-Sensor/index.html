Kidbright Sensor

1 ทำความรู้จักเซนเซอร์บนบอร์ด KidBright

บอร์ด KidBright มีเซนเซอร์พื้นฐานติดตั้งมาให้ 2 ตัว คือ :

  • เซนเซอร์วัดอุณหภูมิ
  • เซนเซอร์วัดแสง

ตำแหน่งเซนเซอร์บนบอร์ด

3 การหา "ค่ากลาง" ( Threshold ) เพื่อตั้งเงื่อนไข

เพื่อให้ระบบแยกแยะได้ว่าตอนไหนคือ "กลางวัน" หรือ "กลางคืน" เราต้องหาค่ากลางมาเป็นเกณฑ์ตัดสิน :

  • บันทึกค่า จดค่าที่อ่านได้ตอนปล่อยให้แสงสว่างปกติ และตอนใช้มือบังเซนเซอร์ให้มืด
  • คำนวณ
    นำค่าทั้งสองมาหาค่าเฉลี่ยด้วยสูตร :
    ( สว่าง ) + ( มืด ) 2
  • ตัวอย่าง : ถ้าสว่างได้ 80 และมืดได้ 10 ค่ากลางคือ
    ( 80 + 10 ) 2
    = 45
